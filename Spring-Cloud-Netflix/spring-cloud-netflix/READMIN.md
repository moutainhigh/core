
# Spring-Cloud-Config

注册中心地址
- http://localhost:9090


运行程序之间必须先启动 kafka,而kafka需要zookeeper,所以必须启动zookeeper + kafka


配置中心 config 端口号是9091


## 启动过程
- 第一步：启动 kafka + zookeeper
- 第二步：访问 http://localhost:9091/user-service.properties
